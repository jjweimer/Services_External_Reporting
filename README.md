# Services_External_Reporting
App for UCSD Library Service Statistics external reporting

## Use and Access
The app can be accessed by visiting http://lib-shiny.ucsd.edu/qualtrics-external-reporting (May require UCSD VPN). Usage requires access to the service statistics qualtrics export. Documentation for acquiring this export can be found within the app. 

## Dependencies
- shiny
- lubridate
- dplyr
- tidyr
- writexl
- shinycssloaders
- bslib


To install all dependencies:
```
install.packages(c("shiny","lubridate","dplyr","tidyr","shinycssloaders","bslib","writexl"))
```
